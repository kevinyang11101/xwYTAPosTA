# 健身房管理系统 java66

## 项目概述

本项目是基于 Spring Boot + Vue 的前后端分离的健身房管理系统设计实现。系统采用 Java 作为后端开发语言，Vue.js 构建前端界面，致力于为健身房提供一个高效、易用的管理平台。

## 技术栈

- 后端：Spring Boot
- 前端：Vue.js

## 功能模块

### 系统角色

系统定义了以下角色：

- 管理员
- 用户
- 教练

### 主要功能

#### 管理员

- 管理用户信息
- 管理教练信息
- 管理健身课程
- 管理健身预约
- 管理课程预约
- 管理器材租借与归还
- 管理预约取消

#### 教练

- 操作健身课程
- 管理健身预约
- 管理课程预约

#### 用户

- 查询健身预约
- 查询课程预约
- 查询器材租借与归还
- 设定个人系统
